# select 属性

- ✅v-model: 当前选中的数据
- ✅placeholder: 占位符
- ✅height: 当设置了该属性后表示菜单的最大高度
- ✅width: select 组件宽度
- ✅size: select 大小
- ✅multiple：多选
- ✅tabindex：html 属性
- ✅theme：主题，mask
- ✅disabled：禁用状态

# select 事件

- ✅input(value)：当选中选项时
- ✅change(value): 当 v-model 值发生改变时

# option 属性

- ✅label：展示内容
- ✅value: 对应的值
- ✅disabled: 禁用
- ✅tabindex：html 属性

# option 插槽

- ✅default：label
